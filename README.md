# ICU - I See You!
ICU is a HexChat script that relays messages between channels, across many servers.

# Installation
In HexChat, click on HexChat -> Load Plugin or Script, and find the icu.py file. That's it!

# Usage
To use ICU, just type /icu to see how to use it:

/ICU <(add|del) (to|from) #channel> Relay messages between channels

So you see, you'll have to be physically in the channels in order to relay any messages, because the script waits for "Channel Message" and "Your Message" events, which will only fire if you're in channels!
